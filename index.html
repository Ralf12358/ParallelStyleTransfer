    • Parallel Style Transfer

    Jason Israel and Michael Kamm, Going for Real-Time style transfer

    Project maintained by jaisrael Hosted on GitHub Pages — Theme by mattgraham

    Summary

    We would like to be able to transfer the style of one image to another. We are going to do this on the NVIDIA GPUs in the lab. Ideally, we would like to achieve real-time analysis.

    Background

    The Challenge

    The most significant part of the challenge comes from achieving a real-time execution. To help meet our goals, we can reduce the quality of the images, essentially decreasing the size of the problem.

    Resources

    The base algorithms that we will use come from two papers. The first describes "Image Analogies" http://www.mrl.nyu.edu/publications/image-analogies/analogies-300dpi.pdf. The second describes "Parallel Texture Synthesis" http://research.microsoft.com/en-us/um/people/hoppe/paratexsyn.pdf

    Goals

    We would really like to be able to make this happen in real time. If we achieve this, we can just show a base image applied to a video. If not, the best we can do is show speedup realtive to a serial solution.

    Platform Choice

    We are choosing to use the NVIDIA GPUs in the lab. Like the second assignment, this project involves math-intensive image manipulation.
