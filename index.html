    Jason Israel and Michael Kamm, Going for Real-Time style transfer

    Project maintained by jaisrael Hosted on GitHub Pages — Theme by mattgraham

    Summary

    We would like to be able to transfer the style of one image to another. We are going to do this on the NVIDIA GPUs in the lab. Ideally, we would like to achieve real-time analysis.

    Background

    The core idea of our project is best described by the term "Image Analogy". An image analogy takes an image (a), a styled version of the first image (A'), and applies the style to a third image (B), to create the result (B'). In the form of an analogy, this is A : A' :: B : B'. Our current starter code executing a similar algorithm currently takes a few seconds on a high definition image.

    Since we are looking to have this execute in real-time, and specifically for videos, we may spend time looking into caching results for the previous frame. The logic here is that Since there are many frames per second, there is not a huge change between frames, making it possible to use previous work.

    There are also preexisting techniques for parallel texture synthesis. The current implementation described in our research is real-time as well.

    The Challenge

    The most significant part of the challenge comes from achieving a real-time execution. To help meet our goals, we can reduce the quality of the images, essentially decreasing the size of the problem. Also, our current starter code is written in MATLAB, so we need to take the time to translate it into C++.

    Resources

    The base algorithms that we will use come from two papers. The first describes "Image Analogies" http://www.mrl.nyu.edu/publications/image-analogies/analogies-300dpi.pdf. The second describes "Parallel Texture Synthesis" http://research.microsoft.com/en-us/um/people/hoppe/apptexsyn.pdf. We have also found a paper detailing a CUDA specific approach http://download.springer.com/static/pdf/45/art%253A10.1007%252Fs00371-012-0701-4.pdf?auth66=1396815636_90dc75f6739fe23fb6a179e7de74a43e&ext=.pdf.

    Goals

    We would really like to be able to make this happen in real time. If we achieve this, we can just show a base image applied to a video. If not, the best we can do is show speedup relative to a serial solution.

    Platform Choice

    We are choosing to use the NVIDIA GPUs in the lab. Like the second assignment, this project involves math-intensive image manipulation.
